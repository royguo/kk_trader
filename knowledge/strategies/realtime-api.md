# 实时行情 API 参考

> 经过实际验证的免费行情数据接口，供每日分析使用。
> **核心原则：每次获取价格必须双源交叉验证，偏差 >1% 时触发警告。**

## ✅ 主力源：腾讯财经 (qt.gtimg.cn)

**状态**：已验证可用（2026-04-10）

### 接口地址

```
http://qt.gtimg.cn/q={代码1},{代码2},...
```

### 代码格式

| 市场 | 前缀 | 示例 | 说明 |
|------|------|------|------|
| 沪市 A 股 | `sh` | `sh600809` | 上交所股票 |
| 深市 A 股 | `sz` | `sz000021` | 深交所股票 |
| 港股 | `hk` | `hk01810` | 港交所股票 |
| 美股 | `us` | `usAAPL` | 纳斯达克/纽交所 |
| 上证指数 | `sh` | `sh000001` | 上证综指 |
| 深证指数 | `sz` | `sz399001` | 深证成指 |
| 恒生指数 | `hk` | `hkHSI` | 恒生指数 |

### 特点

- ✅ 无需认证（无 API Key / Token）
- ✅ 无需 Referer
- ✅ 支持批量查询（逗号分隔）
- ✅ 全市场覆盖（A 股 + 港股 + 美股 + 指数）
- ⚠️ 编码为 GBK，需 `decode('gbk')`

### 返回字段（`~` 分隔）

| 索引 | 字段 | 说明 |
|------|------|------|
| 0 | 市场代码 | 1=沪, 51=深, 100=港, 200=美 |
| 1 | 名称 | 股票/指数名称 |
| 2 | 代码 | 原始代码 |
| **3** | **当前价** | **⭐ 实时价格** |
| 4 | 昨收价 | 前一交易日收盘价 |
| 5 | 今开价 | 当日开盘价 |
| 30 | 时间 | 数据时间戳 |
| 31 | 涨跌额 | 当日涨跌 |
| 32 | 涨跌幅 | 当日涨跌% |
| 33 | 最高价 | 当日最高 |
| 34 | 最低价 | 当日最低 |

## ✅ 验证源：Yahoo Finance API

**状态**：已验证可用（2026-04-10）

### 接口地址

```
https://query1.finance.yahoo.com/v8/finance/chart/{symbol}?interval=1d&range=1d
```

### 代码格式

| 市场 | 格式 | 示例 |
|------|------|------|
| 深市 A 股 | `{code}.SZ` | `000021.SZ` |
| 沪市 A 股 | `{code}.SS` | `600809.SS` |
| 港股 | `{code}.HK` | `1810.HK` |
| 美股 | `{ticker}` | `AAPL` |
| 恒生指数 | `^HSI` | `^HSI` |

### 特点

- ✅ 无需认证
- ✅ 返回标准 JSON
- ✅ 全球市场覆盖
- ⚠️ 不支持批量（需逐个请求）
- ⚠️ 价格在 `chart.result[0].meta.regularMarketPrice`

---

## 🔧 使用方式

```bash
# 查询全部持仓 + 指数（双源验证）
python3 scripts/fetch_quotes.py

# 自定义查询（仅腾讯源）
python3 scripts/fetch_quotes.py usAAPL usNVDA hk00700
```

### 输出中的验证标记

| 标记 | 含义 |
|------|------|
| ✅ | 双源价格一致（偏差 ≤ 1%） |
| ⚠️ | 双源价格有偏差（> 1%），需人工确认 |
| 🅰️ | 仅腾讯源可用 |
| 🅱️ | 仅雅虎源可用 |
| ❌ | 双源均失败 |
