# 实时行情 API 参考

> 经过实际验证的免费行情数据接口，供每日分析使用。

## ✅ 主力 API：腾讯财经 (qt.gtimg.cn)

**状态**：已验证可用（2026-04-10）

### 接口地址

```
http://qt.gtimg.cn/q={代码1},{代码2},...
```

### 代码格式

| 市场 | 前缀 | 示例 | 说明 |
|------|------|------|------|
| 沪市 A 股 | `sh` | `sh600809` | 上交所股票 |
| 深市 A 股 | `sz` | `sz000021` | 深交所股票 |
| 港股 | `hk` | `hk01810` | 港交所股票 |
| 美股 | `us` | `usAAPL` | 纳斯达克/纽交所 |
| 上证指数 | `sh` | `sh000001` | 上证综指 |
| 深证指数 | `sz` | `sz399001` | 深证成指 |
| 创业板指 | `sz` | `sz399006` | 创业板指数 |
| 恒生指数 | `hk` | `hkHSI` | 恒生指数 |

### 特点

- ✅ **无需认证**：不需要 API Key、Token
- ✅ **无需 Referer**：直接 curl 即可
- ✅ **支持批量**：逗号分隔多个代码
- ✅ **全市场覆盖**：A 股 + 港股 + 美股 + 指数
- ⚠️ **编码为 GBK**：解析时需 `decode('gbk')` 或 `iconv -f gbk -t utf-8`

### 返回数据格式

返回 JavaScript 变量赋值字符串，字段以 `~` 分隔：

```
v_sz000021="51~深科技~000021~28.31~27.70~28.21~524487~...~20260410120548~0.61~2.20~28.66~28.01~..."
```

**关键字段索引**（通用于所有市场）：

| 索引 | 字段 | 说明 |
|------|------|------|
| 0 | 市场代码 | 1=沪市, 51=深市, 100=港股, 200=美股 |
| 1 | 名称 | 股票/指数名称 |
| 2 | 代码 | 原始代码 |
| **3** | **当前价** | **⭐ 最新成交价（实时）** |
| 4 | 昨收价 | 前一交易日收盘价 |
| 5 | 今开价 | 当日开盘价 |
| 6 | 成交量 | 成交股数 |
| 30 | 时间 | 数据时间戳 |
| 31 | 涨跌额 | 当日涨跌金额 |
| 32 | 涨跌幅 | 当日涨跌百分比 |
| 33 | 最高价 | 当日最高 |
| 34 | 最低价 | 当日最低 |

### 快速使用

```bash
# 命令行直接查询
curl -s "http://qt.gtimg.cn/q=sz000021,sh600809,hk01810,usAAPL" | iconv -f gbk -t utf-8

# 使用项目脚本（推荐）
python3 scripts/fetch_quotes.py              # 查询全部持仓+指数
python3 scripts/fetch_quotes.py usAAPL usNVDA  # 自定义查询
```

## ❌ 已验证不可用的 API

### 新浪财经 (hq.sinajs.cn)
- **状态**：服务端返回 `Forbidden`（2026-04-10 测试）
- 即使带 Referer 头也被拦截，可能已对服务器 IP 段封禁
- 备注：该接口在浏览器环境中可能仍可用

### 东方财富 (push2.eastmoney.com)
- **状态**：部分可用
- A 股可用（`secid=0.{sz代码}` 或 `1.{sh代码}`），返回 JSON
- 港股、美股 secid 格式不稳定，经常返回空数据
- 不推荐作为主力 API
