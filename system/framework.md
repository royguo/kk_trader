# 投研分析框架

> 本文档定义了 KK Trader 的核心分析方法论，是所有投资决策的底层逻辑支撑。

## 分析层次（自上而下）

### 1. 宏观层 — 方向判断

| 维度 | 关注点 | 数据来源 |
|------|--------|---------|
| 全球流动性 | 美联储/欧央行/日央行政策、利率走势 | 央行声明、经济数据 |
| 地缘政治 | 中美关系、区域冲突、贸易政策 | 新闻、政策文件 |
| 中国宏观 | GDP、PMI、社融、CPI/PPI | 统计局、央行 |
| 汇率 | 人民币/美元、美元指数 | 外汇市场 |
| 大宗商品 | 原油、黄金、铜、铁矿石 | 期货市场 |

### 2. 行业层 — 赛道选择

- 产业政策与监管动态
- 行业景气度周期定位
- 供需格局变化
- 技术变革影响

### 3. 个股层 — 标的筛选

- **基本面**：盈利能力、成长性、估值水平、资产负债表健康度
- **技术面**：趋势、支撑/阻力、量价关系、关键均线
- **资金面**：北向资金、主力资金流向、融资融券变化
- **催化剂**：业绩预告、政策利好、行业事件

## 决策矩阵

```
信号强度 = 宏观方向(30%) + 行业景气(25%) + 个股质量(25%) + 技术时机(20%)
```

- **强买入**：信号强度 ≥ 80，且无重大风险因素
- **买入**：信号强度 ≥ 65
- **持有观望**：信号强度 50-65
- **减仓**：信号强度 < 50 或出现风险信号
- **清仓**：信号强度 < 35 或触发止损

## 多市场特殊考量

### A 股
- T+1 交易制度，涨跌停板限制
- 政策驱动特征明显
- 关注北向资金动向

### 港股
- T+0 交易，无涨跌停
- 受 A 股和美股双重影响
- 流动性分化严重，关注成交量

### 美股
- T+0 交易，盘前盘后交易
- 关注美联储政策和经济数据
- 期权市场对股价影响大
