# 投研分析框架

> 本文档定义了 KK Trader 的核心分析方法论，是所有投资决策的底层逻辑支撑。

## 分析层次（自上而下）

### 1. 宏观层 — 方向判断（决定"是否该在市场里"）

> 📌 宏观层是**最高优先级**。避险信号可覆盖一切个股看多逻辑。

#### 大宗资产"温度计"（每日必查）

| 指标 | 正常 | ⚠️ 避险信号 | 数据来源 |
|------|------|-----------|---------|
| 美元指数(DXY) | 平稳/走弱 | 快速走强 → 资金回流美元 | 财经网站 |
| 国际原油(WTI/Brent) | 供需平衡 | 暴涨(地缘驱动) → 滞胀风险 | 期货市场 |
| 黄金(XAUUSD) | 温和 | 暴涨 → 极端避险 | 期货市场 |
| 白银(XAGUSD) | 跟随黄金 | 金银比剧变 → 情绪切换 | 期货市场 |

#### 中美经济数据（月度追踪）

| 数据 | 来源 | 利好 | 利空 |
|------|------|------|------|
| 中国 CPI/PPI | 统计局（每月中旬） | 温和通胀(1-3%) | 通缩(<0)或高通胀(>5%) |
| 中国社融/M2 | 央行（每月中旬） | 信贷扩张，超预期 | 信贷收缩，不及预期 |
| 中国失业率 | 统计局（每月中旬） | 稳定或下降 | 上升，尤其青年失业率 |
| 美国 CPI/核心CPI | 劳工部（每月中旬） | 趋势下降 | 超预期（压制降息） |
| 美国非农就业 | 劳工部（首周五） | 健康增长 | 急剧恶化=衰退信号 |
| 美联储利率 | FOMC（约6周一次） | 鸽派/降息 | 鹰派/推迟降息 |

#### 其他宏观维度

| 维度 | 关注点 | 数据来源 |
|------|--------|---------|
| 全球流动性 | 美联储/欧央行/日央行政策、利率走势 | 央行声明、经济数据 |
| 地缘政治 | 中美关系、区域冲突、贸易政策 | 新闻、政策文件 |
| 汇率 | 人民币/美元、美元指数 | 外汇市场 |

### 2. 行业层 — 赛道选择

- 产业政策与监管动态
- 行业景气度周期定位
- 供需格局变化
- 技术变革影响

### 3. 个股层 — 标的筛选

- **基本面**：盈利能力、成长性、估值水平、资产负债表健康度
- **技术面**：趋势、支撑/阻力、量价关系、关键均线
- **资金面**：北向资金、主力资金流向、融资融券变化
- **催化剂**：业绩预告、政策利好、行业事件
- **事件共振**：近期重大事件（科技大会/财报/政策）与个股逻辑的契合度

### 4. 事件催化层 — 短期机会捕捉（新增）

> 📌 事件催化是独立于常规自上而下分析的"机会扫描器"

#### 每日事件日历扫描

| 维度 | 内容 |
|------|------|
| 时间窗口 | 未来 7 天内的重大事件 |
| 事件类型 | 科技大会、财报季、政策发布、央行决议、行业展会 |
| 分析方法 | 事件 → 受益产业链 → 具体标的 → 是否在关注池/持仓中 |

#### 事件驱动操作框架

| 阶段 | 时间窗口 | 动作 |
|------|---------|------|
| 预热期 | 事件前 5-10 天 | 识别受益标的，研究逻辑，轻仓试探 |
| 事件期 | 当天 ± 1天 | 观察市场反应，确认逻辑兑现情况 |
| 消化期 | 事件后 3-5 天 | 逻辑兑现→可加仓；证伪→果断退出 |

#### 主动发现机制

- 分析过程中发现的**关注池外**优质标的，应主动推荐
- 推荐格式：标的名称 + 发现路径 + 核心逻辑 + 风险提示 + 建议仓位
- 最高优先级：**为 KK 提供最大收益、最低风险的投资建议**

## 决策矩阵

### 第一层：宏观环境 → 仓位水位（参见 decision-model.md）

### 第二层：个股信号评估

```
个股信号 = 行业景气(0.30) + 个股基本面(0.30) + 技术面/资金面(0.25) + 事件催化(0.15)
```

- **强买入**：信号强度 ≥ 80，且宏观环境允许
- **买入**：信号强度 ≥ 65
- **持有观望**：信号强度 50-65
- **减仓**：信号强度 < 50 或宏观转向避险
- **清仓**：逻辑被证伪，或宏观进入强避险模式

## 多市场特殊考量

### A 股
- T+1 交易制度，涨跌停板限制
- 政策驱动特征明显
- 关注北向资金动向

### 港股
- T+0 交易，无涨跌停
- 受 A 股和美股双重影响
- 流动性分化严重，关注成交量

### 美股
- T+0 交易，盘前盘后交易
- 关注美联储政策和经济数据
- 期权市场对股价影响大
