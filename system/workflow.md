# 每日工作流程

> 本文档定义了每日例行投研工作的标准流程。

## 📋 日常流程

### Phase 1: 上下文加载（每次对话开始时）

1. 读取 `README.md` — 了解系统定位与当前状态
2. 读取 `portfolio/current.md` — 加载当前持仓
3. 读取最近 1-3 天的 `daily/` 记录 — 回顾近期分析与决策

### Phase 2: 市场扫描

1. **隔夜市场回顾**
   - 美股三大指数收盘情况
   - 欧洲主要指数表现
   - 亚太市场动态
2. **当日市场概览**
   - A 股/港股开盘情况与盘中走势
   - 板块轮动与资金流向
3. **宏观事件检查**
   - 重要经济数据发布
   - 央行政策动态
   - **🔴 地缘政治风险扫描（独立一阶）**：
     - 主要冲突区（中东/台海/俄乌等）最新军事/外交进展
     - 关键运输通道（霍尔木兹海峡、曼德海峡、马六甲）通航情况
     - 停火/制裁协议到期日与谈判进展
     - 主要大国新外交声明或制裁行动
     - 当前地缘阶段判断（🟢缓和/🟡博弈/🟠升级/🔴危机）
4. **大宗商品与汇率**
   - 原油、黄金、白银走势
   - 美元指数、人民币汇率
   - 避险信号组合判断（参见 decision-model.md）
5. **📅 事件日历扫描**（新增）
   - 未来 7 天内的重大国际/国内事件（科技大会、财报、政策、央行等）
   - 识别事件与持仓/关注标的的共振点
   - 预热期标的筛选（事件前 5-10 天提前布局分析）

### Phase 3: 持仓评估

对每个持仓标的进行评估：
- 当日表现与原因分析
- 是否触发风控规则
- 持有逻辑是否仍然成立
- 是否需要调整仓位

### Phase 4: 输出建议

生成结构化建议报告：
```
## 今日建议 [日期]

### 宏观环境判断
[避险信号级别 + 允许仓位水位]

### 市场判断
[对整体市场的观点]

### 📅 近期重大事件
| 日期 | 事件 | 影响板块 | 关联标的 |
|------|------|---------|---------|

### 持仓操作建议
| 标的 | 当前仓位 | 建议操作 | 目标仓位 | 理由 |
|------|---------|---------|---------|------|

### 关注池标的更新
[关注标的的最新动态和买入时机判断]

### 🔍 新发现推荐（池外标的）
[分析过程中发现的、不在关注池中但值得考虑的标的]
[必须说明：发现路径、核心逻辑、风险提示]

### 风险提示
[需要警惕的风险因素]
```

### Phase 5: 记录归档

将当日分析写入 `daily/YYYY/MM/DD.md`，包含：
- 市场概况
- 操作记录（实际执行的交易）
- 分析与建议
- 复盘笔记

### Phase 6: 复盘学习（周末或重要事件后）

- 对比预测与实际结果
- 提炼经验教训 → 更新 `knowledge/lessons.md`
- 发现新的分析模式 → 更新 `knowledge/` 相关文档
- 能力提升记录 → 更新 `system/evolution-log.md`

### Phase 7: 会话结束知识提炼 🧬

> 当 KK 发出结束信号（"结束"、"再见"、"下班"等）时，自动执行以下操作：

1. **提炼新认知**：回顾本次对话中出现的新观点、新信息、新判断
2. **更新投资理念**：如有新的理念讨论或迭代，更新 `knowledge/strategies/philosophy.md`
3. **记录经验教训**：如有错误或新发现，更新 `knowledge/lessons.md`
4. **更新知识库**：将有价值的行业/宏观信息沉淀到 `knowledge/` 相关文档
5. **归档日志**：确保当日分析已写入 `daily/YYYY/MM/DD.md`
6. **更新进化日志**：如有重要能力/认知升级，更新 `system/evolution-log.md`
7. **提交变更**：将所有更新通过 git 提交

## 🚨 事件驱动响应

当发生以下事件时，即使不是例行分析时间也应触发分析：
- 持仓标的重大异动（涨跌幅 > 5%）
- 重要政策发布
- 黑天鹅事件
- 上级（KK）紧急咨询

## 📝 沟通规范

### 给出建议时
- 明确区分「事实」和「判断」
- 量化风险（止损位、最大回撤预期）
- 给出确信度评级（高/中/低）
- 无法确认的数据主动标注并请求上级补充

### 接收上级指令时
- 理解指令意图
- 综合自身分析给出独立判断
- 如有分歧，明确表达并说明理由
- 最终尊重上级决策
