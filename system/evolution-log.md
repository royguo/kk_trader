### 2026-04-10 | v0.4.0 | 宏观避险框架建立

**变更类型**：能力升级 + 认知升级

**内容**：
- 确立投资理念三「宏观避险优先原则」— 大宗+经济数据决定仓位水位
- 建立完整的宏观监测指标体系：美元指数、原油、黄金、白银 + CPI/PPI/社融/失业率
- 设计四级避险信号组合判断：🔴强避险/🟠中度/🟡关注/🟢风险偏好
- 重构评估权重为「两层决策」：宏观层决定仓位上限 → 个股层在水位内选股
- 升级分析框架宏观层，从概略描述升级为具体指标+阈值+数据源

**核心变化**：
- 旧：宏观环境作为个股评估的一个权重维度(30%)
- 新：宏观环境作为**系统级开关**，直接决定仓位水位上限，优先于个股判断
- 本质：先判断"该不该在市场里"，再判断"买什么"

### 2026-04-10 | v0.3.0 | 投资理念体系建立

**变更类型**：认知升级

**内容**：
- 确立核心投资理念「前瞻性决策原则」— 只看未来走势，不纠结当前盈亏
- 创建 `knowledge/strategies/philosophy.md` 作为理念专属文档
- 重构风控规则：移除止损/止盈，建立回撤预警体系（10%/20%/30%三级）
- 新增工作流 Phase 7：会话结束知识提炼（自动化理念迭代）
- 更新 README 体现持续进化定位

**核心理念变化**：
- 旧：亏损8%→自动减仓，亏损12%→无条件清仓
- 新：回撤10%→分析预警，回撤20%→深度评估，回撤30%→强制论证
- 本质：用「理性的重新评估」替代「机械的止损」，完全废除自动卖出

**教训**：
- 投资理念应该是一个持续构建的过程，不是一次性设定的参数
- 好的理念需要与风控纪律调和，而不是对立
